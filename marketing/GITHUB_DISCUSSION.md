# Wallet HD Derivation Kit 1.0: seven native languages, one conformance contract

ROYO STUDIOS is releasing Wallet HD Derivation Kit 1.0 for Swift, JavaScript/React Native, Python, Rust, Go, Dart/Flutter, Kotlin/JVM/Android, and the `wallethd` CLI.

The interesting part is not the chain count; it is the release gate. The same public requests run in seven native implementations, and any address disagreement fails conformance. The package is offline-only, keeps private exports explicit, documents standards boundaries, and ships public vectors, fuzz/property tests, benchmarks, SBOMs, checksums, and provenance.

Docs: https://devdasx.github.io/wallet-hd-derivation-kit/
Source: https://github.com/devdasx/wallet-hd-derivation-kit

The project has not yet received an independent audit. Feedback on API ergonomics, vectors, threat modeling, and additional chain schemes is welcome.
