# Third-party notices

This MIT-licensed project depends on third-party cryptographic and utility libraries. Their source, licenses, copyright notices, and exact resolved versions are available through the committed `Package.resolved`, `package-lock.json`, `Cargo.lock`, `go.sum`, `pubspec.lock`, Gradle dependency report, and Python package metadata.

Notable implementations include Bitcoin Core secp256k1 via `secp256k1.swift`, CryptoSwift, Swift Crypto, Noble Cryptography, `@scure/bip39`, `bip-utils`, RustCrypto, `k256`, `ed25519-dalek`, btcsuite, `go-bip39`, `blockchain_utils`, Bouncy Castle, and bitcoinj. Distribution workflows generate CycloneDX/SPDX SBOMs containing the complete versioned set for each release.
